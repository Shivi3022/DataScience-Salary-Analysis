# Clean Data

**File:** `ds_salaries_clean.csv`  
**Records:** 607 rows × 13 columns  
Generated by `notebooks/Eda_notebook.ipynb` after feature engineering.

## Added Columns

| Column | Description |
|---|---|
| `experience_label` | Entry-Level / Mid-Level / Senior / Executive |
| `employment_label` | Full-Time / Part-Time / Contract / Freelance |
| `company_size_label` | Small / Medium / Large |
| `work_setting` | On-Site / Hybrid / Remote |
| `salary_band` | `<50K` / `50K-100K` / `100K-150K` / `150K-200K` / `200K+` |
| `is_us_company` | 1 if US company, 0 otherwise |
| `exp_order` | Numeric sort key for experience (1=Entry → 4=Executive) |

> Use `ds_salaries_clean.csv` for Tableau / Power BI dashboards.
