# Raw Data

**File:** `ds_salaries.csv`  
**Source:** [Kaggle — Data Science Job Salaries](https://www.kaggle.com/datasets/ruchi798/data-science-job-salaries)  
**Records:** 607 rows × 12 columns  
**Period:** 2020–2022

## Columns

| Column | Type | Description |
|---|---|---|
| `work_year` | int | Year salary was paid |
| `experience_level` | str | EN / MI / SE / EX |
| `employment_type` | str | FT / PT / CT / FL |
| `job_title` | str | Role name |
| `salary` | int | Raw salary (local currency) |
| `salary_currency` | str | Currency code |
| `salary_in_usd` | int | Salary converted to USD |
| `employee_residence` | str | Employee's country code |
| `remote_ratio` | int | 0 = On-site, 50 = Hybrid, 100 = Remote |
| `company_location` | str | Company's country code |
| `company_size` | str | S / M / L |
