# Dashboard

**File:** `Data_Science_Salary_Dashboard.pbix`  
**Tool:** Microsoft Power BI Desktop

## How to open

1. Download and install [Power BI Desktop](https://powerbi.microsoft.com/desktop/) (free)
2. Open `Data_Science_Salary_Dashboard.pbix`
3. If data doesn't load, go to **Home → Transform Data → Data Source Settings** and update the path to `../data/clean/ds_salaries_clean.csv`

## Charts included

- KPI cards — Mean, Median, Max salary
- Salary distribution histogram
- Median salary by experience level
- Salary trend 2020–2022
- Work setting breakdown
- Top 10 job titles by median salary
- Company size comparison
